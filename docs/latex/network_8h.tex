\doxysubsection{Detailed Description}
Network abstraction layer using Valve\textquotesingle{}s Game\+Networking\+Sockets. 

This file provides a high-\/level C++ wrapper around Game\+Networking\+Sockets, offering both client and server functionality with callback-\/based event handling.
