\doxysubsection{Detailed Description}
Base network class with common functionality. 

Provides shared initialization and shutdown logic for both \doxylink{class_network_client}{Network\+Client} and \doxylink{class_network_server}{Network\+Server} classes. Manages the underlying ISteam\+Networking\+Sockets interface.

\begin{DoxyNote}{Note}
This class should not be instantiated directly. Use \doxylink{class_network_client}{Network\+Client} or \doxylink{class_network_server}{Network\+Server} instead. 
\end{DoxyNote}
