\hypertarget{index_intro_sec}{}\doxysection{\texorpdfstring{Introduction}{Introduction}}\label{index_intro_sec}
R-\/\+Type is a multiplayer networked game inspired by the classic arcade shooter. This project implements a client-\/server architecture using Valve\textquotesingle{}s Game\+Networking\+Sockets library for reliable UDP communication.\hypertarget{index_arch_sec}{}\doxysection{\texorpdfstring{Architecture Overview}{Architecture Overview}}\label{index_arch_sec}
The project is organized into three main components\+:\hypertarget{index_core_subsec}{}\doxysubsection{\texorpdfstring{Core Library (r-\/type\+\_\+core)}{Core Library (r-\/type\+\_\+core)}}\label{index_core_subsec}
The core library contains shared code used by both the client and server\+:
\begin{DoxyItemize}
\item \doxylink{class_network_base}{Network\+Base}, \doxylink{class_network_client}{Network\+Client}, \doxylink{class_network_server}{Network\+Server} -\/ Network abstraction layer over Game\+Networking\+Sockets
\item \doxylink{class_message_buffer}{Message\+Buffer} -\/ Binary serialization/deserialization utilities
\item \doxylink{struct_client_state}{Client\+State}, \doxylink{struct_game_state_message}{Game\+State\+Message}, \doxylink{struct_update_state_message}{Update\+State\+Message} -\/ Message structures
\item Shared constants and configuration values
\end{DoxyItemize}\hypertarget{index_client_subsec}{}\doxysubsection{\texorpdfstring{Client Application (r-\/type\+\_\+client)}{Client Application (r-\/type\+\_\+client)}}\label{index_client_subsec}
The game client handles\+:
\begin{DoxyItemize}
\item Rendering using raylib
\item Player input processing
\item Network communication with the server
\item Local game state interpolation
\end{DoxyItemize}\hypertarget{index_server_subsec}{}\doxysubsection{\texorpdfstring{Server Application (r-\/type\+\_\+server)}{Server Application (r-\/type\+\_\+server)}}\label{index_server_subsec}
The dedicated server handles\+:
\begin{DoxyItemize}
\item \doxylink{class_client}{Client} connection management
\item Game state synchronization
\item Broadcasting updates to all connected clients
