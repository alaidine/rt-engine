This project uses\+:
\begin{DoxyItemize}
\item {\bfseries{Raylib 5.\+0}}\+: Graphics and game framework
\item {\bfseries{Sol2 3.\+3.\+0}}\+: Modern C++ Lua binding library ~\newline

\item {\bfseries{Lua 5.\+4.\+7}}\+: Embedded scripting language (precompiled libraries in {\ttfamily lua/})
\item {\bfseries{Box2D 3.\+1.\+1}}\+: 2D physics engine
\end{DoxyItemize}

{\bfseries{Core Library}}\+: All shared dependencies are linked into {\ttfamily r-\/type\+\_\+core} static library, which is then used by both client and server executables.

{\bfseries{Automatic Fetching}}\+: CMake automatically downloads raylib, sol2, and box2d.
