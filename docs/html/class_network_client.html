Detailed Description

Network client for connecting to a game server.

Provides client-side networking functionality including connection management, message sending/receiving, and connection statistics. Uses a callback-based architecture for handling network events.

Example Usage:
NetworkClient client;
client.SetOnConnected([](const uint8_t* data, size_t size) {
// Handle connection established
});
client.SetOnMessageReceived([](const uint8_t* data, size_t size) {
// Handle incoming message
});
if (client.Connect("127.0.0.1", 42042)) {
while (client.IsConnected()) {
client.Poll();
// Game loop...
}
}
