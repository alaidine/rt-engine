Detailed Description

Binary buffer for message serialization and deserialization.

Provides a simple interface for reading and writing binary data in network messages. Uses little-endian byte order.

Writing Example:
MessageBuffer buf;
buf.WriteU8(MSG_SPAWN_INFO);
buf.WriteU32(clientId);
buf.WriteI32(x);
buf.WriteI32(y);
network->SendMessage(buf.Data(), buf.Size(), true);
