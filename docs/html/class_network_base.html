Detailed Description

Base network class with common functionality.

Provides shared initialization and shutdown logic for both NetworkClient and NetworkServer classes. Manages the underlying ISteamNetworkingSockets interface.

Note
This class should not be instantiated directly. Use NetworkClient or NetworkServer instead.
