Detailed Description

Network server for hosting game sessions.

Provides server-side networking functionality including listening for connections, managing multiple clients, and broadcasting messages. Uses a callback-based architecture for handling network events.

Example Usage:
NetworkServer server;
server.SetOnNewConnection([](HSteamNetConnection conn, uint32_t id) -> bool {
return true; // Accept all connections
});
server.SetOnClientConnected([](HSteamNetConnection conn, uint32_t id) {
// Send initial game state to new client
});
server.SetOnMessageReceived([](HSteamNetConnection conn, uint32_t id,
const uint8_t* data, size_t size) {
// Handle client message
});
if (server.Start(42042)) {
while (running) {
server.Poll();
// Game loop...
}
server.Stop();
}
