enum MessageType : uint8_t

Message type identifiers for the network protocol.

Each message sent over the network begins with a MessageType byte to identify the payload format.

Enumerator
MSG_SPAWN_INFO 

Server → Client: Initial spawn position and client ID.

MSG_UPDATE_STATE 

Client → Server: Player position and missiles update.

MSG_GAME_STATE 

Server → Client: Complete game state with all players.
